# test_verbal_remarks

Проект автоматизированного тестирования с использованием **Pytest** и генерацией отчётов с помощью **Allure**.

## Описание

Это тестовый проект для демонстрации и отработки verbal remarks (вербальных замечаний/комментариев) в тестах.  
Проект включает:
- Тесты в директории `tests/`
- Утилиты и вспомогательный код в `utils/`
- Конфигурацию Pytest в `conftest.py`
- Автоматический скрипт запуска тестов и генерации Allure-отчёта (`run_tests_and_allure.bat`)
- Генерацию отчётов Allure (результаты в `allure-results/`, готовые отчёты в `reports/verbal/`)

Проект написан на Python и использует фреймворк Pytest для запуска тестов.

## Требования

- Python 3.8+
- Установленные зависимости (см. ниже)
